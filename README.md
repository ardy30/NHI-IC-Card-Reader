# NHI-IC-Card-Reader
Simple card reader for National Health Insurance IC card.

    This is C++ program.
    User can use this program to read the
    NHI IC card's basic information.
    Include :
    NHI IC Card ID
    Name 
    ID 
    Gender 
    Issue date


Need to add `winscard.lib` into additional dependencies in VS2013.
